### Run Locally

To run **Autofix** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/autofix.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/autofix.git
```
